## Description 

  *The what, why, and how:* 
  
  ğŸ” A CLI content management system for managing a company's employees using node, inquirer, and MySQL. Employee Tracker is a CLI applicaiton for building and maintaining companies employee data base. Built on MySQL and leveraging modularity in the structure of Employee Tracker, it can scale appropriatley with a companies growth without reconfiguring the core structure of the internal employee data base. This application allows the user to insert new data such as departments, roles, employees to data base.  In addition, user can view, update and delete company's data on all departments, role and employees that we install in data base.
